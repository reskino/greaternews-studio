# Content Exports

This folder is where GreaterNews daily pack exports go.

The web app (GreaterNews Studio) downloads files for local saving:

- `greaternews_YYYY-MM-DD.md` — the full day pack: story menu, selected story details, and every output format (Facebook, X, Instagram, WhatsApp, video script, news card).
- `posted_log.md` — the running log of published stories, used to make sure a story is never repeated.

Move downloaded files into this folder to keep the editorial trail in one place.
