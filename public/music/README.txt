GreaterNews — your own royalty-free music tracks
================================================

1. Drop audio files here (mp3, m4a, ogg or wav).
2. List each one in tracks.json, e.g.:

   [
     { "file": "afro-groove.mp3", "label": "Afro Groove" },
     { "file": "cinematic-rise.mp3", "label": "Cinematic Rise" }
   ]

   - "file" must match the file name exactly (case-sensitive).
   - "label" is what shows in the Video sound dropdown.

3. Commit + push so it deploys (git add public/music && git commit && git push).

They then appear in the studio under "My music", can be previewed, and are mixed
under the voiceover just like the built-in moods. Tracks loop to fill the video.

ONLY use music you have the right to use commercially. Good free sources:
  - Pixabay Music        (https://pixabay.com/music/)  — free, no attribution
  - YouTube Audio Library (in YouTube Studio)           — free, no attribution
Avoid random tracks from the internet or streaming rips.
